Project 1 of 6

Fresco

Aplikasi yang dapat mendeteksi kematangan buah yang gambar yang didapat. Untuk pengambilan gambar bisa dilakukan dengan dua cara, yaitu mengambil gambar dari galeri dan mengambil foto dengan kamera secara langsung.

Untuk membuat aplikasi ini saya berkolaborasi dengan Machine Learning Engineer, dimana Machine Learning Engineer bertugas untuk membuat model untuk mendeteksi kematangan buah dengan tflite, lalu saya sebagai Android Developer bertugas untuk membuat Aplikasi Android dengan bahasa pemrograman Kotlin dan menggunakan model yang telah dibuat Team Machine Learning ke dalam aplikasi ini.

Pada pembuatan aplikasi, mengalami kesulitan untuk membuat pendeteksi kematangan buah berjalan di Aplikasi Android. Namun, dengan melihat berbagai referensi yang tersedia seperti youtube dan github, saya berhasil mengimplementasikan deteksi buah-buahan pada Aplikasi Android.

Teknologi yang digunakan

  • Kotlin
  • TensorFlow Lite

Aplikasi yang menampilkan daftar rumah sakit rujukan covid di Indonesia. Menampilkan jumlah ketersediaan kamar berdasarkan kelas kamar di tiap rumah sakit.

Project 5 of 6

Shamo App

Aplikasi E-Commerce yang menampilkan dan menjual beberapa produk sepatu. Pada aplikasi ini calon pembeli dapet memanfaatkan fitur chat untuk menanyakan ketersediaan barang serta menanyakan ukuran sepatu yang tersedia. Pengguna dapat menambahkan produk sepatu ke dalam daftar favorit.

Aplikasi ini terintegrasi dengan SQL Database sebagai backend yang dibangun dengan framework Laravel versi 8. Database berisi data akun pengguna aplikasi, daftar produk, daftar gambar, dan daftar kategori produk. Lalu, aplikasi ini juga terintegrasi dengan Firebase sebagai Realtime-Database yang digunakan untuk fitur chat pada aplikasi ini.

  • Flutter
  • Provider State Management
  • Firebase
  • Laravel 8
Februari - Maret 2022

Project 6 of 6

Smart Bali Backpacker

Smart Bali Backpacker merupakan aplikasi berbasis Android yang dibuat untuk mempromosikan tempat wisata di Bali sebagai upaya untuk meningkatkan jumlah wisatawan baik lokal maupun global di Pulau Bali.

Aplikasi ini dibangun dengan bahasa pemrograman Kotlin yang terintegrasi dengan API yang dibangun dengan framework Laravel dan backend nosql yaitu Firebase untuk autentikasi. Pada aplikasi ini user dapat melihat daftar tempat wisata di Bali disertai dengan informasi detail mengenai tempat wisata tersebut. Kemudian, user dapat memanfaatkan fitur pemanduan perjalanan menuju tempat wisata yang akan dituju dengan Google Maps yang terpasang pada aplikasi. Selain itu, pada aplikasi ini user dapat melakukan chatting untuk berkomunikasi dengan user lainnya.

  • Kotlin
  • Firebase
  • Retrofit
  • Room Database
  • MVVM Architecture
Oktober 2021 - Oktober 2022
